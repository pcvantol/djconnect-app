# DJConnect App 3.2.3

- Track Insight voelt rustiger en consistenter op iPhone, met normale iOS-titels,
  betere kaartvolgorde en metric-blokken die netjes meeschalen met de tekst.
- Track Insight delen heeft een duidelijkere sheet, scrollt beter op kleine
  iPhones en annuleert geanimeerde exports direct.
- De Track Insight visualizer rendert lichter en stopt animaties zodra je het
  scherm verlaat.
