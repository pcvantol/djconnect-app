# DJConnect App 3.2.17

## Gewijzigd

- DJConnect gebruikt nu de canonieke Home Assistant API-routes onder
  `/api/djconnect/v1/...`.
- Pairing QR-codes en deep-links geven nu
  `pair_path=/api/djconnect/v1/pair` door.
- API- en releasedocumentatie beschrijft nu overal de v1-routeprefix.

## Opgelost

- Regressietests toegevoegd zodat Apple-clientroutes op de v1-prefix blijven.
- Door de server teruggegeven image-, TTS- en audio-URL's blijven ongewijzigd
  aan de clientkant.
