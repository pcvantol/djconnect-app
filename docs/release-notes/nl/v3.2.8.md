# DJConnect App 3.2.8

- DJConnect heeft nu een gedeelde lokalisatiebasis voor Engels, Nederlands,
  Duits, Frans en Spaans in de Apple-clients.
- Koppelen en Home Assistant API-fouten gebruiken nu een centrale
  gebruikersmelding, waaronder verkeerd app-type, ongeldige koppelcode, verlopen
  auth en niet-geconfigureerde responses.
- Protocolwaarden zoals `client_type`, endpointpaden en JSON-keys blijven
  ongewijzigd en worden niet vertaald.
- Een lokalisatievalidator controleert nu dat iedere gedeelde key in alle vijf
  ondersteunde talen bestaat en dat format-placeholders gelijk blijven.
- De bijdrage-richtlijnen vereisen nu dat user-facing Apple-clienttekst in
  dezelfde pull request voor iedere ondersteunde taal wordt toegevoegd.
