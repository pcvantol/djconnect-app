# DJConnect App 3.2.14

## Toegevoegd

- VibeCast-feedondersteuning toegevoegd op iOS en macOS, inclusief
  gestructureerde facts, trivia en side bubbles zolang het VibeCast-scherm actief
  is.
- WebSocket fast-path ondersteuning toegevoegd voor de route
  `djconnect/vibecast`, met HTTP fallback wanneer de route niet beschikbaar is.
- Automatische Track Insight-analyse toegevoegd terwijl VibeCast streamt, zodat
  de huidige track en elke volgende track klaarstaan zonder telkens op Analyze
  Now te tikken.

## Opgelost

- Release-CI gerepareerd voor systemen waarop Nederlands de standaardtaal van de
  app is.
