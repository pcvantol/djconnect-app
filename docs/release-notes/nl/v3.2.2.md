# DJConnect App 3.2.2

## Nieuw

- Track Insight heeft nu uitgebreid delen, widgets en iPhone Live Activity / Dynamic Island visuals.
- Ask DJ en Track Insight zijn afgestemd op het Home Assistant 3.2.2 Music DNA-contract.
- Lokale Home Assistant-verbindingen kunnen optioneel de native WebSocket fast path gebruiken voor snellere acties, met HTTP als fallback.
- iOS krijgt Home Screen quick actions voor Ask DJ en Track Insight.

## Let op

- HTTP blijft de vaste fallback voor alle ondersteunde acties.
- Widgets gebruiken alleen veilige, zichtbare Track Insight-velden; private Home Assistant-URL's, entity ID's, player ID's, tokens en diagnostics worden niet gedeeld.
