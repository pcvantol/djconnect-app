# DJConnect App 3.2.23

## Cambios

- Las tarjetas de Music DNA en iPhone son mas altas y hacen scroll interno
  cuando el contenido no cabe en pantallas pequenas o en horizontal.
- Apple Watch ahora tiene Descubrir con lista de recomendaciones, explicacion
  detallada y Reproducir ahora mediante el proxy del iPhone.
- Track Insight usa el mismo selector flotante de Mood que Ask DJ.
- Los juegos usan un area de juego mas baja en iPad horizontal para que los
  botones debajo del juego sigan visibles.
- Compartir Track Insight usa una vista previa mas ajustada y mas espacio
  inferior para los botones de accion.
- El flujo de release comprueba de forma mas estricta el commit exacto y las
  notas de version vinculadas a la version.

## Corregido

- El icono de copiar logs en iOS permanece blanco en vez de heredar el tinte
  morado.
- La respuesta haptica ahora tambien funciona en modo demo en un Apple Watch
  real.
