# DJConnect App 3.2.25

## Ajoute

- Now Playing sur iOS propose maintenant un bouton Queue direct a cote de shuffle.

## Modifie

- Le widget Now Playing utilise maintenant un placeholder DJConnect music/play
  quand la pochette manque, au lieu d'un bloc gris.
- La Live Activity de l'ecran verrouille et Dynamic Island correspondent mieux
  au widget Now Playing, avec les couleurs Mood, la pochette, une pochette
  compacte dans Dynamic Island et les memes visuels de fallback.
