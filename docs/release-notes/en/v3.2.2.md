# DJConnect App 3.2.2

## Highlights

- Track Insight now supports richer sharing, widgets, and iPhone Live Activity / Dynamic Island visuals.
- Ask DJ and Track Insight are aligned with the Home Assistant 3.2.2 Music DNA contract.
- Local Home Assistant connections can use an optional native WebSocket fast path for lower-latency actions, with HTTP fallback preserved.
- iOS adds Home Screen quick actions for Ask DJ and Track Insight.

## Notes

- HTTP remains the canonical fallback for all supported actions.
- Widgets use safe, visible Track Insight fields only; private Home Assistant URLs, entity IDs, player IDs, tokens, and diagnostics are not shared.
