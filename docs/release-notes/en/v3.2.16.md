# DJConnect App 3.2.16

## Added

- Added Music DNA export and import in iOS/macOS Settings.
- Music DNA export now downloads the server-built JSON backup from Home
  Assistant and saves it through the native Apple save/share sheet.
- Music DNA import previews a selected JSON backup, shows profile statistics,
  and uploads it to Home Assistant only while paired and connected.

## Changed

- Polished Music DNA, Discover, Ask DJ, Track Insight, hero sizing, toasts, and
  five-language translations.
- Updated release hygiene so all app-facing strings are reviewed in Dutch,
  English, German, Spanish, and French.

## Fixed

- Fixed Music DNA opt-in prompts appearing when the backend state is unknown.
- Fixed Music DNA export/import auth and backend errors to use the normal
  DJConnect recovery messages.
