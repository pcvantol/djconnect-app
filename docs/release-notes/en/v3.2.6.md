# DJConnect App 3.2.6

- QR pairing on iPhone now explains camera access before iOS asks for permission.
- Home-screen quick actions and widgets now open the matching DJConnect page more reliably.
- Demo mode refreshes Now Playing, Queue, Playlists, Track Insight and Ask DJ widgets consistently.
- Track Insight and VibeCast visuals animate more smoothly while music is playing.
- Music DNA on Apple Watch now matches iOS/macOS icon style and button alignment.
- Track Insight sharing keeps exported media at the correct size while scaling only the preview.
