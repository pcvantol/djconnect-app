# DJConnect App 3.2.7

- Music DNA uses the same outline heart icon across Apple Watch, iPhone/iPad and macOS.
- Music DNA no longer flips back to the old enabled state when a stale profile refresh arrives right after changing the setting.
- Pairing now gives clearer messages for wrong Home Assistant app-type selection, invalid URLs and local-development ngrok URLs.
- Ask DJ keeps a freshly received answer visible when a later history sync arrives.
- Track Insight sends the correct client type to Home Assistant and localizes client-type errors.
- Output-device choices on iOS and macOS now feel closer to the Apple Watch selector style.
