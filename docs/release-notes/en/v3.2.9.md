# DJConnect App 3.2.9

- Ask DJ chat clearing now immediately empties the local chat on iOS, macOS and
  watchOS after a successful Home Assistant clear response.
- Clear-history requests now include the canonical Apple client identity fields
  so Home Assistant can clear the correct device/user context.
- iPad and macOS layouts use more of the available canvas for Track Insight,
  Music DNA, local games, About, Legal and Privacy.
- Popover-style sheets, onboarding and QR scanning feel larger and more natural
  on iPad.
- The macOS splash screen now includes a richer vector music/DJ visual.
- Several small UI polish fixes landed for permissions, logs, More, output
  devices, Track Insight export and Music DNA cards.
