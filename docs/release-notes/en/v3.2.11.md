# DJConnect App 3.2.11

- Ask DJ assistant answers now tint with the selected mood on iPhone, iPad, Mac,
  and Apple Watch.
- Ask DJ uses backend metadata for generated text and mood, without guessing
  from visible text or marking fixed fallback answers as AI-generated.
- Music DNA now shows richer profile details, including average mood, energy
  profile, danceability, intensity, and recent signals.
- Track Insight now uses the same client mood, particles, and color styling in
  live view, sharing, and AirPlay/VibeCast output.
- Apple Watch now resets Track Insight correctly when a new song starts and has
  a tighter Music DNA presentation with a watch-specific animation.
- Track Insight progress is clearer and moves more smoothly with the current
  song.
- Ask DJ follow-up buttons, such as more from the same artist, now start a new
  Ask DJ request correctly.
