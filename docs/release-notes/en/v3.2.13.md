# DJConnect App 3.2.13

## Added

- Added VibeCast feed support on iOS and macOS, including structured facts,
  trivia and side bubbles while the VibeCast screen is active.
- Added WebSocket fast-path support for the `djconnect/vibecast` route, with
  HTTP fallback when the route is unavailable.
- Added automatic Track Insight analysis while VibeCast is streaming, so the
  current track and each next track stay ready without tapping Analyze Now.
