# DJConnect App 3.2.18

## Added

- About now includes a check-for-updates button that can show localized release
  notes for one or more patch updates.
- VibeCast can show the Home Assistant genre badge in the top trailing corner
  when the backend sends `context.genre_badge`.
- iPhone now advertises landscape support explicitly.

## Changed

- Music DNA and Track Insight cards keep a more comfortable size on iPhone by
  scrolling horizontally per row instead of shrinking.
- Discover, More, Ask DJ, pairing, Track Insight, and Music DNA received iPhone
  layout and copy polish.

## Fixed

- Pairing QR scanning now uses continuous focus and exposure.
- Ask DJ input focus is steadier and no longer publishes SwiftUI state during a
  view update.
- Xcode warning cleanup for app icons, watch proxy payload parsing, and camera
  configuration.
